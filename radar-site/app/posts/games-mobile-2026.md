# Games mobile estão ficando cada vez mais parecidos com jogos de console

O mercado de games mobile deixou de ser apenas jogos simples para passar o tempo.

Hoje, celulares rodam títulos pesados, recebem controles Bluetooth, suportam telas com alta taxa de atualização e competem diretamente pela atenção de quem antes jogava apenas em console ou PC.

Isso não significa que o celular vai substituir tudo, mas mostra uma mudança clara: o smartphone virou uma plataforma gamer séria.

Para muita gente, o primeiro videogame moderno já está no bolso.
